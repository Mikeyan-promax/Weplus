# WePlus 校园助手开发记录

## 项目概述
WePlus 是一个基于 RAG（检索增强生成）技术的智能校园助手系统，旨在为学生提供便捷的校园生活服务。

## 技术栈
- **前端**: React + TypeScript + Vite
- **后端**: Python FastAPI
- **AI模型**: DeepSeek API
- **向量数据库**: Chroma/FAISS (计划中)
- **样式**: CSS3 + 响应式设计
